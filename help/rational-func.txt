Rational Functions - ArithFuns Package

Create and manipulate rational functions (ratios of polynomials).

CONSTRUCTOR:
  RatFunc(P, Q)             - Create P(x)/Q(x) from polynomials

PROPERTIES:
  RatFuncNumer(R)           - Get numerator polynomial
  RatFuncDenom(R)           - Get denominator polynomial
  Get(R, "type")            → "ratfunc"

EVALUATION:
  RatFuncEval(R, x)         - Evaluate R(x) (undefined at poles)

ARITHMETIC:
  RatFuncAdd(R, S)          - Add: R + S
  RatFuncSub(R, S)          - Subtract: R - S
  RatFuncMul(R, S)          - Multiply: R · S
  RatFuncDiv(R, S)          - Divide: R / S
  RatFuncSimplify(R)        - Cancel common factors

CALCULUS:
  RatFuncDer(R)             - Derivative (quotient rule)

ANALYSIS:
  RatFuncPoles(R)           - Find poles (roots of denominator)
  RatFuncZeros(R)           - Find zeros (roots of numerator)
  RatFuncAsymp(R)           - Asymptotic behavior analysis

PARTIAL FRACTIONS:
  PartialFrac(R)            - Partial fraction decomposition
  PartialFracSteps(R)       - Show decomposition steps

EXAMPLES:
  P := Poly({1, 2})         # 1 + 2x
  Q := Poly({1, 0, 1})      # 1 + x²
  R := RatFunc(P, Q)        # (1 + 2x)/(1 + x²)
  
  RatFuncEval(R, 0)         → 1
  RatFuncEval(R, 1)         → 3/2
  
  # Partial fractions of 1/(x² - 1)
  F := RatFunc(Poly({1}), Poly({-1, 0, 1}))
  PartialFrac(F)
  → { terms: {...}, polynomial: Poly({0}) }

PARTIAL FRACTION OUTPUT:
  The decomposition returns:
  {
    polynomial: Poly(...),    # Polynomial part (if deg(P) ≥ deg(Q))
    terms: [
      {coeff: a, root: r, power: 1},   # a/(x - r)
      {coeff: b, root: s, power: 2},   # b/(x - s)²
      ...
    ]
  }

NOTES:
  - Simplify automatically reduces to lowest terms
  - Poles are values where denominator is zero
  - Partial fractions require denominator to factor over rationals
  - For irreducible quadratic factors, terms use (Ax+B)/(x²+px+q) form
